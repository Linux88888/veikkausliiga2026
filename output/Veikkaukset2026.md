## ✏️ Lisää oma veikkauksesi

Haluatko osallistua? Lisää oma veikkauksesi muokkaamalla `scripts/config.py`-tiedostoa.

**1.** Avaa `scripts/config.py` omalla koodieditorillasi.

**2.** Etsi `PARTICIPANTS`-lista ja lisää oma lohkosi:

```python
{
    "name": "Sinun Nimi",
    "standings_prediction": [
        "HJK",           # Veikkaat 1. sijaksi
        "KuPS",          # 2. sija
        "FC Inter",      # 3. sija
        "SJK",           # 4. sija
        "Ilves",         # 5. sija
        "FC Lahti",      # 6. sija
        "FF Jaro",       # 7. sija
        "VPS",           # 8. sija
        "IFK Mariehamn", # 9. sija
        "IF Gnistan",    # 10. sija
        "AC Oulu",       # 11. sija
        "TPS",           # 12. sija
    ],
    "scorers_prediction": [
        "Plange, Luke",         # 1. maalintekijä
        "Karjalainen, Rasmus",  # 2. maalintekijä
        "Odutayo, Colin",       # 3. maalintekijä
        "Coffey, Ashley",       # 4. maalintekijä
        "Moreno, Jaime",        # 5. maalintekijä
    ],
},
```

**3.** Aja `python scripts/main.py` — pisteet päivittyvät automaattisesti! 🚀
