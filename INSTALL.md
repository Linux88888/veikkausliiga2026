# INSTALL.md

## Quick Installation
- Use the following command to clone the repository:
    ```bash
    git clone https://github.com/Linux88888/veikkausliiga2026.git
    ```
- Change into the project directory:
    ```bash
    cd veikkausliiga2026
    ```
- Install the required Python packages:
    ```bash
    pip install -r requirements.txt
    ```

## Configuration
- To configure the application, modify the `config.py` file:
  - Set up the teams:
    ```python
    TEAMS = ['Team A', 'Team B', 'Team C']
    ```
  - Add players for each team:
    ```python
    PLAYERS = {
        'Team A': ['Player 1', 'Player 2'],
        'Team B': ['Player 3', 'Player 4'],
    }
    ```
  - Define the predicted order:
    ```python
    PREDICTED_ORDER = ['Team A', 'Team B', 'Team C']
    ```

## Schedule Setup for Linux/Mac Crontab
- You can set up a cron job to run the script automatically:
    ```bash
    crontab -e
    # Add the following line to run the script daily at Midnight
    0 0 * * * python /path/to/veikkausliiga2026/script.py
    ```

## GitHub Actions Automatic Updates
- This project includes GitHub Actions for automatic updates. The workflow is defined in `.github/workflows/auto-update.yml`, which checks for updates daily and pulls changes if necessary.

## Generated Files Description
- This section explains the files generated by the application:
  - `output.txt`: Contains the results of the latest simulation.
  - `logs/`: A directory containing log files for each run, useful for debugging.

## Troubleshooting
- Common Issues:
  - **ModuleNotFoundError**:
    - Solution: Ensure all dependencies are installed. Run `pip install -r requirements.txt` again.
  - **Connection Issues**:
    - Solution: Check your internet connection and ensure your API keys in `config.py` are correct.
  - **File Update Problems**:
    - Solution: Make sure you have write permissions for the directory where the output files are stored.
